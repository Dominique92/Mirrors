Le but est d'envoyer des lasers pour découvrir quelles cases contiennent des miroirs.

Cliquer sur les bougies pour allumer les lasers.

Cliquer sur les cases pour marquer vos suppositions de miroirs.

Double cliquer sur les cases qui ne contiennent pas de miroir pour en vérifier le contenu.

Vous avez gagné quand vous avez découvert tous les emplacement de miroir sans ouvrir leur cases.
